# data-analytics-portfolio
## Profil
Data Analyst spécialisé en reporting et analyse de données.
J’ai travaillé sur des données réelles (stocks hospitaliers 2020–2024) avec Power BI, SQL et Talend.
Je transforme des données brutes en tableaux de bord clairs pour aider à la prise de décision.

- Gestion de données multi-années
